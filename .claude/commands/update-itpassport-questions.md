# ITパスポート問題更新スキル

ITパスポートの3カテゴリ（ストラテジ系・マネジメント系・テクノロジ系）の問題を新たに10問ずつ生成し、JSONファイルを更新する。

## 手順

### 1. 参照資料の確認（任意）
CLAUDE.md に記載されている参照資料URLが利用可能であれば、WebFetchで取得して問題作成の参考にすること。
取得できない場合はスキップして問題作成を続行すること。

### 2. 問題の生成ルール

**共通ルール：**
- 各カテゴリ10問ずつ作成する
- 過去問を参考にした**予想問題**として作成する（実際の過去問をそのまま転用しない）
- 問題文・選択肢・解説はすべて日本語で記述する
- ヒントは150文字程度、解説は200文字程度とする
- `answer` は `"choice1"` 〜 `"choice4"` のいずれかとし、正解が特定の選択肢番号に偏らないようにする

**JSONフォーマット（厳守）：**
```json
{
  "questions": [
    {
      "question": "問題文",
      "answer": "choice1",
      "hint": "ヒントのテキスト（150文字程度）",
      "explanation": "解説のテキスト（200文字程度）",
      "choice1": "選択肢1",
      "choice2": "選択肢2",
      "choice3": "選択肢3",
      "choice4": "選択肢4",
      "scope": "カテゴリ名（例：ストラテジ系）",
      "text": "出題範囲（例：企業活動）"
    }
  ]
}
```

### 3. ストラテジ系（strategy_questions.json）

- 企業活動

- ビジネスインダストリ
- システム戦略
- システム企画
